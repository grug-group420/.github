# Contributing to grug-group420

Thanks for wanting to contribute! 🦴

## The Grug Way

1. **Simple > Clever**
2. **Less code > More code**  
3. **Working > Perfect**
4. **Ship fast**

## How to Contribute

1. Fork the repo
2. Make your change (keep it small)
3. Open PR
4. Done

## PR Checklist

- [ ] Code works
- [ ] Code is simple
- [ ] No new dependencies

## What We Accept

✅ Bug fixes
✅ Documentation  
✅ Code deletion
✅ Simple features

## What We Don't

❌ Complex abstractions
❌ Many dependencies
❌ "Clever" solutions

---

**Ship it!** 🚀
